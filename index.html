使用说明

  • 此工具用于登记你的课表并依此生成值日表，请务必正确、完整地填写所有课表。包括自习课。
  • 登记完成后，请点击 "导出课程文件" 导出课表 JSON 文件，并保证命名为 学号-姓名.json
  • 然后，将导出的课表 Pull Request 到指定仓库
  • 以后若要编辑课表，可以先点击 "导入已导出的课程" 来导入之前导出的课表
